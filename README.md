# more_order_app

注文が苦手な人のためのアプリ。
もっと簡単に注文できるようになることを目指す。

## Requirements

Flutterのインストールなどを行う。
[参考](https://zenn.dev/kazutxt/books/flutter_practice_introduction/viewer/06_chapter1_environment)

## Getting Started

For Mac Users

```sh
make setup_all
```

For Windows Users

```sh
make setup
```

## Usage

- VS Codeで開く
- デバイスを選択する
- lib/main.dartを開く
- デバッグ起動する
